жҺўзҙўзұ»еһӢзі»з»ҹ

TypeScript зҡ„иҜӯиЁҖжңҚеҠЎ

TypeScript зҡ„иҜӯиЁҖжңҚеҠЎпјҢд№ҹиў«з§°дёә tsserverпјҢжҸҗдҫӣдәҶеҗ„з§ҚеҠҹиғҪпјҢдҫӢеҰӮй”ҷиҜҜжҠҘе‘ҠгҖҒиҜҠж–ӯгҖҒдҝқеӯҳж—¶зј–иҜ‘гҖҒйҮҚе‘ҪеҗҚгҖҒи·іиҪ¬еҲ°е®ҡд№үгҖҒиЎҘе…ЁеҲ—иЎЁгҖҒзӯҫеҗҚеё®еҠ©зӯүгҖӮе®ғдё»иҰҒз”ұйӣҶжҲҗејҖеҸ‘зҺҜеўғ (IDE) дҪҝз”ЁжқҘжҸҗдҫӣ IntelliSense ж”ҜжҢҒгҖӮе®ғдёҺ Visual Studio Code ж— зјқйӣҶжҲҗпјҢе№¶з”ұ Conquer of Completion (Coc) зӯүе·Ҙе…·дҪҝз”ЁгҖӮ

ејҖеҸ‘дәәе‘ҳеҸҜд»ҘеҲ©з”Ёдё“з”Ё API е№¶еҲӣе»әиҮӘе·ұзҡ„иҮӘе®ҡд№үиҜӯиЁҖжңҚеҠЎжҸ’д»¶жқҘеўһејә TypeScript зј–иҫ‘дҪ“йӘҢгҖӮиҝҷеҜ№дәҺе®һзҺ°зү№ж®Ҡзҡ„ linting еҠҹиғҪжҲ–еҗҜз”ЁиҮӘе®ҡд№үжЁЎжқҝиҜӯиЁҖзҡ„иҮӘеҠЁе®ҢжҲҗзү№еҲ«жңүз”ЁгҖӮ

зҺ°е®һдё–з•Ңдёӯзҡ„иҮӘе®ҡд№үжҸ’д»¶зҡ„дёҖдёӘзӨәдҫӢжҳҜвҖқtypescript-styled-pluginвҖқпјҢе®ғдёәж ·ејҸз»„д»¶дёӯзҡ„ CSS еұһжҖ§жҸҗдҫӣиҜӯжі•й”ҷиҜҜжҠҘе‘Ҡе’Ң IntelliSense ж”ҜжҢҒгҖӮ

жңүе…іжӣҙеӨҡдҝЎжҒҜе’Ңеҝ«йҖҹе…Ҙй—ЁжҢҮеҚ—пјҢжӮЁеҸҜд»ҘеҸӮиҖғ GitHub дёҠзҡ„е®ҳж–№ TypeScript Wikiпјҡ https://github.com/microsoft/TypeScript/wiki/

з»“жһ„зұ»еһӢ

TypeScript еҹәдәҺз»“жһ„зұ»еһӢзі»з»ҹгҖӮиҝҷж„Ҹе‘ізқҖзұ»еһӢзҡ„е…је®№жҖ§е’Ңзӯүж•ҲжҖ§з”ұзұ»еһӢзҡ„е®һйҷ…з»“жһ„жҲ–е®ҡд№үеҶіе®ҡпјҢиҖҢдёҚжҳҜз”ұе…¶еҗҚз§°жҲ–еЈ°жҳҺдҪҚзҪ®еҶіе®ҡпјҢеҰӮ C# жҲ– C зӯүдё»иҰҒзұ»еһӢзі»з»ҹдёӯйӮЈж ·гҖӮ

TypeScript зҡ„з»“жһ„зұ»еһӢзі»з»ҹжҳҜеҹәдәҺ JavaScript зҡ„еҠЁжҖҒйёӯеӯҗзұ»еһӢзі»з»ҹеңЁиҝҗиЎҢж—¶зҡ„е·ҘдҪңж–№ејҸиҖҢи®ҫи®Ўзҡ„гҖӮ

д»ҘдёӢзӨәдҫӢжҳҜжңүж•Ҳзҡ„ TypeScript д»Јз ҒгҖӮжӯЈеҰӮжӮЁжүҖи§ӮеҜҹеҲ°зҡ„пјҢвҖңXвҖқе’ҢвҖқYвҖқе…·жңүзӣёеҗҢзҡ„жҲҗе‘ҳвҖқaвҖқпјҢе°Ҫз®Ўе®ғд»¬е…·жңүдёҚеҗҢзҡ„еЈ°жҳҺеҗҚз§°гҖӮзұ»еһӢз”ұе…¶з»“жһ„еҶіе®ҡпјҢеңЁиҝҷз§Қжғ…еҶөдёӢпјҢз”ұдәҺз»“жһ„зӣёеҗҢпјҢеӣ жӯӨе®ғд»¬жҳҜе…је®№дё”жңүж•Ҳзҡ„гҖӮ

TypeScript зҡ„еҹәжң¬жҜ”иҫғи§„еҲҷ

TypeScript жҜ”иҫғиҝҮзЁӢжҳҜйҖ’еҪ’зҡ„пјҢе№¶еңЁд»»дҪ•зә§еҲ«еөҢеҘ—зҡ„зұ»еһӢдёҠжү§иЎҢгҖӮ

еҰӮжһңвҖқYвҖқиҮіе°‘е…·жңүдёҺвҖқXвҖқзӣёеҗҢзҡ„жҲҗе‘ҳпјҢеҲҷзұ»еһӢвҖқXвҖқдёҺвҖқYвҖқе…је®№гҖӮ

еҮҪж•°еҸӮж•°жҢүзұ»еһӢиҝӣиЎҢжҜ”иҫғпјҢиҖҢдёҚжҳҜжҢүеҗҚз§°иҝӣиЎҢжҜ”иҫғпјҡ

еҮҪж•°иҝ”еӣһзұ»еһӢеҝ…йЎ»зӣёеҗҢпјҡ

жәҗеҮҪж•°зҡ„иҝ”еӣһзұ»еһӢеҝ…йЎ»жҳҜзӣ®ж ҮеҮҪж•°зҡ„иҝ”еӣһзұ»еһӢзҡ„еӯҗзұ»еһӢпјҡ

е…Ғи®ёдёўејғеҮҪж•°еҸӮж•°пјҢеӣ дёәиҝҷжҳҜ JavaScript дёӯзҡ„еёёи§ҒеҒҡжі•пјҢдҫӢеҰӮдҪҝз”Ё вҖңArray.prototype.map()вҖңпјҡ

еӣ жӯӨпјҢд»ҘдёӢзұ»еһӢеЈ°жҳҺжҳҜе®Ңе…Ёжңүж•Ҳзҡ„пјҡ

жәҗзұ»еһӢзҡ„д»»дҪ•йҷ„еҠ еҸҜйҖүеҸӮж•°йғҪжҳҜжңүж•Ҳзҡ„пјҡ

зӣ®ж Үзұ»еһӢзҡ„д»»дҪ•еҸҜйҖүеҸӮж•°еңЁжәҗзұ»еһӢдёӯжІЎжңүеҜ№еә”зҡ„еҸӮж•°йғҪжҳҜжңүж•Ҳзҡ„е№¶дё”дёҚжҳҜй”ҷиҜҜпјҡ

е…¶дҪҷеҸӮж•°иў«и§Ҷдёәж— йҷҗзі»еҲ—зҡ„еҸҜйҖүеҸӮж•°пјҡ

еҰӮжһңйҮҚиҪҪзӯҫеҗҚдёҺе…¶е®һзҺ°зӯҫеҗҚе…је®№пјҢеҲҷе…·жңүйҮҚиҪҪзҡ„еҮҪж•°жңүж•Ҳпјҡ

еҰӮжһңжәҗеҸӮж•°е’Ңзӣ®ж ҮеҸӮж•°еҸҜиөӢеҖјз»ҷи¶…зұ»еһӢжҲ–еӯҗзұ»еһӢпјҲBivariance еҸҢеҸҳпјүпјҢеҲҷеҮҪж•°еҸӮж•°жҜ”иҫғжҲҗеҠҹгҖӮ

жһҡдёҫдёҺж•°еӯ—е…·жңүеҸҜжҜ”жҖ§е’Ңжңүж•ҲжҖ§пјҢеҸҚд№ӢдәҰз„¶пјҢдҪҶжҜ”иҫғдёҚеҗҢжһҡдёҫзұ»еһӢзҡ„жһҡдёҫеҖјжҳҜж— ж•Ҳзҡ„гҖӮ

зұ»зҡ„е®һдҫӢйңҖиҰҒеҜ№е…¶з§ҒжңүжҲҗе‘ҳе’ҢеҸ—дҝқжҠӨжҲҗе‘ҳиҝӣиЎҢе…је®№жҖ§жЈҖжҹҘпјҡ

жҜ”иҫғжЈҖжҹҘдёҚиҖғиҷ‘дёҚеҗҢзҡ„з»§жүҝеұӮж¬Ўз»“жһ„пјҢдҫӢеҰӮпјҡ

жіӣеһӢж №жҚ®еә”з”ЁжіӣеһӢеҸӮж•°еҗҺзҡ„з»“жһңзұ»еһӢдҪҝз”Ёе…¶з»“жһ„иҝӣиЎҢжҜ”иҫғпјҢд»…е°ҶжңҖз»Ҳз»“жһңдҪңдёәйқһжіӣеһӢзұ»еһӢиҝӣиЎҢжҜ”иҫғгҖӮ

еҪ“жіӣеһӢжңӘжҢҮе®ҡе…¶зұ»еһӢеҸӮж•°ж—¶пјҢжүҖжңүжңӘжҢҮе®ҡзҡ„еҸӮж•°йғҪе°Ҷиў«и§ҶдёәеёҰжңүвҖқanyвҖқзҡ„зұ»еһӢпјҡ

и®°дҪҸпјҡ

иҜ·жіЁж„ҸпјҢеҪ“еҗҜз”ЁвҖқstrictNullChecksвҖқж—¶пјҢвҖңnullвҖқе’ҢвҖқundefinedвҖқзҡ„еӨ„зҗҶж–№ејҸдёҺвҖқvoidвҖқзұ»дјјпјӣеҗҰеҲҷпјҢе®ғд»¬зұ»дјјдәҺвҖқneverвҖқгҖӮ

зұ»еһӢдҪңдёәйӣҶеҗҲ

еңЁ TypeScript дёӯпјҢзұ»еһӢжҳҜдёҖз»„еҸҜиғҪзҡ„еҖјгҖӮиҜҘйӣҶеҗҲд№ҹз§°дёәзұ»еһӢзҡ„еҹҹгҖӮзұ»еһӢзҡ„жҜҸдёӘеҖјйғҪеҸҜд»Ҙиў«и§ҶдёәйӣҶеҗҲдёӯзҡ„дёҖдёӘе…ғзҙ гҖӮзұ»еһӢе»әз«ӢдәҶйӣҶеҗҲдёӯзҡ„жҜҸдёӘе…ғзҙ еҝ…йЎ»ж»Ўи¶іжүҚиғҪиў«и§ҶдёәиҜҘйӣҶеҗҲзҡ„жҲҗе‘ҳзҡ„зәҰжқҹгҖӮTypeScript зҡ„дё»иҰҒд»»еҠЎжҳҜжЈҖжҹҘе№¶йӘҢиҜҒдёҖз»„жҳҜеҗҰжҳҜеҸҰдёҖз»„зҡ„еӯҗйӣҶгҖӮ

дҫӢеҰӮпјҡ

еңЁдёҠйқўзҡ„зӨәдҫӢдёӯпјҢдҪҝз”Ё as е…ій”®еӯ—е°ҶеҜ№иұЎ x ж–ӯиЁҖдёәзұ»еһӢ XгҖӮиҝҷйҖҡзҹҘ TypeScript зј–иҜ‘еҷЁиҜҘеҜ№иұЎз¬ҰеҗҲжҢҮе®ҡзҡ„зұ»еһӢпјҢеҚідҪҝе®ғе…·жңүзұ»еһӢе®ҡд№үдёӯдёҚеӯҳеңЁзҡ„йҷ„еҠ еұһжҖ§ bгҖӮ

зұ»еһӢж–ӯиЁҖеңЁйңҖиҰҒжҢҮе®ҡжӣҙе…·дҪ“зұ»еһӢзҡ„жғ…еҶөдёӢйқһеёёжңүз”ЁпјҢе°Өе…¶жҳҜеңЁдҪҝз”Ё DOM ж—¶гҖӮдҫӢеҰӮпјҡ

жӯӨеӨ„пјҢзұ»еһӢж–ӯиЁҖ HTMLInputElement з”ЁдәҺе‘ҠиҜү TypeScript getElementById зҡ„з»“жһңеә”иў«и§Ҷдёә HTMLInputElementгҖӮзұ»еһӢж–ӯиЁҖиҝҳеҸҜд»Ҙз”ЁдәҺйҮҚж–°жҳ е°„й”®пјҢеҰӮдёӢйқўдҪҝз”ЁжЁЎжқҝж–Үеӯ—зҡ„зӨәдҫӢжүҖзӨәпјҡ

еңЁжӯӨзӨәдҫӢдёӯпјҢзұ»еһӢ J дҪҝз”ЁеёҰжңүжЁЎжқҝж–Үеӯ—зҡ„жҳ е°„зұ»еһӢжқҘйҮҚж–°жҳ е°„ Type зҡ„й”®гҖӮе®ғеҲӣе»әж–°еұһжҖ§пјҢе№¶еңЁжҜҸдёӘй”®дёҠж·»еҠ  prefix_ пјҢе®ғд»¬еҜ№еә”зҡ„еҖјжҳҜиҝ”еӣһеҺҹе§ӢеұһжҖ§еҖјзҡ„еҮҪж•°гҖӮ

еҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҢеҪ“дҪҝз”Ёзұ»еһӢж–ӯиЁҖж—¶пјҢTypeScript дёҚдјҡжү§иЎҢеӨҡдҪҷзҡ„еұһжҖ§жЈҖжҹҘгҖӮеӣ жӯӨпјҢеҪ“йў„е…ҲзҹҘйҒ“еҜ№иұЎзҡ„з»“жһ„ж—¶пјҢйҖҡеёёжңҖеҘҪдҪҝз”Ёзұ»еһӢеЈ°жҳҺгҖӮ

йқһз©әж–ӯиЁҖ

жӯӨж–ӯиЁҖжҳҜдҪҝз”ЁеҗҺзјҖиЎЁиҫҫејҸ!иҝҗз®—з¬Ұеә”з”Ёзҡ„пјҢе®ғе‘ҠиҜү TypeScript еҖјдёҚиғҪдёә null жҲ–жңӘе®ҡд№үгҖӮ

зҺҜеўғеЈ°жҳҺ

зҺҜеўғеЈ°жҳҺжҳҜжҸҸиҝ° JavaScript д»Јз Ғзұ»еһӢзҡ„ж–Үд»¶пјҢе®ғд»¬зҡ„ж–Үд»¶еҗҚж јејҸдёә.d.ts.. е®ғд»¬йҖҡеёёиў«еҜје…Ҙе№¶з”ЁдәҺжіЁйҮҠзҺ°жңүзҡ„ JavaScript еә“жҲ–еҗ‘йЎ№зӣ®дёӯзҡ„зҺ°жңү JS ж–Үд»¶ж·»еҠ зұ»еһӢгҖӮ

и®ёеӨҡеёёи§Ғзҡ„еә“зұ»еһӢеҸҜд»ҘеңЁд»ҘдёӢдҪҚзҪ®жүҫеҲ°пјҡ https://github.com/DefinitelyTyped/DefinitelyTyped/

еҜ№дәҺжӮЁе®ҡд№үзҡ„зҺҜеўғеЈ°жҳҺпјҢжӮЁеҸҜд»ҘдҪҝз”ЁвҖқдёүж–ңжқ вҖқеј•з”ЁеҜје…Ҙпјҡ

еҚідҪҝеңЁ JavaScript ж–Үд»¶дёӯпјҢжӮЁд№ҹеҸҜд»ҘйҖҡиҝҮ // @ts-check дҪҝз”ЁзҺҜеўғеЈ°жҳҺгҖӮ

declare е…ій”®еӯ—еҸҜд»ҘдёәзҺ°жңүзҡ„ JavaScript д»Јз ҒеҗҜз”Ёзұ»еһӢе®ҡд№үпјҢиҖҢж— йңҖеҜје…Ҙе®ғпјҢдҪңдёәжқҘиҮӘеҸҰдёҖдёӘж–Үд»¶жҲ–е…ЁеұҖзҡ„зұ»еһӢзҡ„еҚ дҪҚз¬ҰгҖӮ

еұһжҖ§жЈҖжөӢе’ҢеӨҡдҪҷеұһжҖ§жЈҖжөӢ

TypeScript еҹәдәҺз»“жһ„зұ»еһӢзі»з»ҹпјҢдҪҶиҝҮеӨҡзҡ„еұһжҖ§жЈҖжҹҘжҳҜ TypeScript зҡ„дёҖдёӘеұһжҖ§пјҢе®ғе…Ғи®ёе®ғжЈҖжҹҘеҜ№иұЎжҳҜеҗҰе…·жңүзұ»еһӢдёӯжҢҮе®ҡзҡ„зЎ®еҲҮеұһжҖ§гҖӮ

дҫӢеҰӮпјҢеңЁе°ҶеҜ№иұЎеӯ—йқўйҮҸиөӢеҖјз»ҷеҸҳйҮҸжҲ–е°Ҷе®ғд»¬дҪңдёәеҸӮж•°дј йҖ’з»ҷеҮҪж•°зҡ„еӨҡдҪҷеұһжҖ§ж—¶пјҢдјҡжү§иЎҢеӨҡдҪҷеұһжҖ§жЈҖжҹҘгҖӮ

ејұзұ»еһӢ

еҪ“дёҖдёӘзұ»еһӢеҸӘеҢ…еҗ«дёҖз»„е…ЁеҸҜйҖүеұһжҖ§ж—¶пјҢиҜҘзұ»еһӢиў«и®ӨдёәжҳҜејұзұ»еһӢпјҡ

еҪ“жІЎжңүйҮҚеҸ ж—¶пјҢTypeScript и®Өдёәе°Ҷд»»дҪ•еҶ…е®№иөӢеҖјз»ҷејұзұ»еһӢжҳҜй”ҷиҜҜзҡ„пјҢдҫӢеҰӮпјҢд»ҘдёӢдјҡеј•еҸ‘й”ҷиҜҜпјҡ

е°Ҫз®ЎдёҚжҺЁиҚҗпјҢдҪҶеҰӮжһңйңҖиҰҒпјҢеҸҜд»ҘдҪҝз”Ёзұ»еһӢж–ӯиЁҖз»•иҝҮжӯӨжЈҖжҹҘпјҡ

жҲ–иҖ…йҖҡиҝҮе°Ҷunknownзҙўеј•зӯҫеҗҚж·»еҠ еҲ°ејұзұ»еһӢпјҡ

дёҘж јзҡ„еҜ№иұЎеӯ—йқўйҮҸжЈҖжөӢ (Freshness)

дёҘж јзҡ„еҜ№иұЎеӯ—йқўйҮҸжЈҖжҹҘпјҲжңүж—¶з§°дёәвҖңж–°йІңеәҰвҖқпјүжҳҜ TypeScript дёӯзҡ„дёҖйЎ№еҠҹиғҪпјҢжңүеҠ©дәҺжҚ•иҺ·еӨҡдҪҷжҲ–жӢјеҶҷй”ҷиҜҜзҡ„еұһжҖ§пјҢеҗҰеҲҷиҝҷдәӣеұһжҖ§еңЁжӯЈеёёз»“жһ„зұ»еһӢжЈҖжҹҘдёӯдјҡиў«еҝҪи§ҶгҖӮ

еҲӣе»әеҜ№иұЎеӯ—йқўйҮҸж—¶пјҢTypeScript зј–иҜ‘еҷЁи®Өдёәе®ғжҳҜвҖңж–°йІңзҡ„вҖқгҖӮ еҰӮжһңе°ҶеҜ№иұЎеӯ—йқўйҮҸеҲҶй…Қз»ҷеҸҳйҮҸжҲ–дҪңдёәеҸӮж•°дј йҖ’пјҢе№¶дё”еҜ№иұЎеӯ—йқўйҮҸжҢҮе®ҡзӣ®ж Үзұ»еһӢдёӯдёҚеӯҳеңЁзҡ„еұһжҖ§пјҢеҲҷ TypeScript е°Ҷеј•еҸ‘й”ҷиҜҜгҖӮ

з„¶иҖҢпјҢеҪ“жү©еұ•еҜ№иұЎж–Үеӯ—жҲ–дҪҝз”Ёзұ»еһӢж–ӯиЁҖж—¶пјҢвҖңж–°йІңж„ҹвҖқе°ұдјҡж¶ҲеӨұгҖӮ

дёӢйқўдёҫдёҖдәӣдҫӢеӯҗжқҘиҜҙжҳҺпјҡ

зұ»еһӢжҺЁж–ӯ

еҪ“еңЁд»ҘдёӢжңҹй—ҙжңӘжҸҗдҫӣжіЁйҮҠж—¶пјҢTypeScript еҸҜд»ҘжҺЁж–ӯзұ»еһӢпјҡ

  • еҸҳйҮҸеҲқе§ӢеҢ–
  • жҲҗе‘ҳеҲқе§ӢеҢ–гҖӮ
  • и®ҫзҪ®еҸӮж•°зҡ„й»ҳи®ӨеҖјгҖӮ
  • еҮҪж•°иҝ”еӣһзұ»еһӢгҖӮ

дҫӢеҰӮпјҡ

TypeScript зј–иҜ‘еҷЁеҲҶжһҗеҖјжҲ–иЎЁиҫҫејҸе№¶ж №жҚ®еҸҜз”ЁдҝЎжҒҜзЎ®е®ҡе…¶зұ»еһӢгҖӮ

жӣҙй«ҳзә§зҡ„жҺЁж–ӯ

еҪ“еңЁзұ»еһӢжҺЁж–ӯдёӯдҪҝз”ЁеӨҡдёӘиЎЁиҫҫејҸж—¶пјҢTypeScript дјҡжҹҘжүҫвҖқжңҖдҪіеёёи§Ғзұ»еһӢвҖқгҖӮдҫӢеҰӮпјҡ

еҰӮжһңзј–иҜ‘еҷЁжүҫдёҚеҲ°жңҖдҪійҖҡз”Ёзұ»еһӢпјҢе®ғе°Ҷиҝ”еӣһиҒ”еҗҲзұ»еһӢгҖӮдҫӢеҰӮпјҡ

TypeScript еҲ©з”ЁеҹәдәҺеҸҳйҮҸдҪҚзҪ®зҡ„вҖқдёҠдёӢж–Үзұ»еһӢвҖқжқҘжҺЁж–ӯзұ»еһӢгҖӮеңЁдёӢйқўзҡ„зӨәдҫӢдёӯпјҢзј–иҜ‘еҷЁзҹҘйҒ“е®ғзҡ„eзұ»еһӢжҳҜMouseEventпјҢеӣ дёәеңЁlib.d.ts ж–Үд»¶дёӯе®ҡд№үдәҶclickдәӢд»¶зұ»еһӢпјҢиҜҘж–Үд»¶еҢ…еҗ«еҗ„з§Қеёёи§Ғ JavaScript жһ„йҖ е’Ң DOM зҡ„зҺҜеўғеЈ°жҳҺпјҡ

зұ»еһӢеҠ е®Ҫ

зұ»еһӢеҠ е®ҪжҳҜ TypeScript е°Ҷзұ»еһӢеҲҶй…Қз»ҷжңӘжҸҗдҫӣзұ»еһӢжіЁйҮҠж—¶еҲқе§ӢеҢ–зҡ„еҸҳйҮҸзҡ„иҝҮзЁӢгҖӮе®ғе…Ғи®ёд»ҺзӘ„еҲ°е®Ҫзҡ„зұ»еһӢпјҢдҪҶеҸҚд№ӢеҲҷдёҚз„¶гҖӮеңЁд»ҘдёӢзӨәдҫӢдёӯпјҡ

TypeScriptж №жҚ®еҲқе§ӢеҢ–жңҹй—ҙжҸҗдҫӣзҡ„еҚ•дёӘеҖјпјҲxпјүпјҢе°Ҷ string иөӢдәҲз»ҷ xпјҢиҝҷжҳҜдёҖдёӘжү©еұ•зҡ„зӨәдҫӢгҖӮ

TypeScript жҸҗдҫӣдәҶжҺ§еҲ¶еҠ е®ҪиҝҮзЁӢзҡ„ж–№жі•пјҢдҫӢеҰӮдҪҝз”ЁвҖқconstвҖқгҖӮ

еёёйҮҸ

еңЁеЈ°жҳҺеҸҳйҮҸж—¶дҪҝз”Ё const е…ій”®еӯ—дјҡеҜјиҮҙ TypeScript дёӯзҡ„зұ»еһӢжҺЁж–ӯиҢғеӣҙжӣҙзӘ„гҖӮ

For example:

йҖҡиҝҮдҪҝз”Ё const еЈ°жҳҺеҸҳйҮҸ xпјҢе…¶зұ»еһӢиў«зј©е°Ҹдёәзү№е®ҡзҡ„ж–Үеӯ—еҖјвҖқxвҖқгҖӮз”ұдәҺ x зҡ„зұ»еһӢиў«зј©е°ҸпјҢеӣ жӯӨеҸҜд»Ҙе°Ҷе…¶иөӢеҖјз»ҷеҸҳйҮҸ y иҖҢдёҚдјҡеҮәзҺ°д»»дҪ•й”ҷиҜҜгҖӮеҸҜд»ҘжҺЁж–ӯзұ»еһӢзҡ„еҺҹеӣ жҳҜеӣ дёә const еҸҳйҮҸж— жі•йҮҚж–°еҲҶй…ҚпјҢеӣ жӯӨе®ғд»¬зҡ„зұ»еһӢеҸҜд»Ҙзј©е°ҸеҲ°зү№е®ҡзҡ„ж–Үеӯ—зұ»еһӢпјҢеңЁжң¬дҫӢдёӯдёәеӯ—йқўйҮҸзұ»еһӢвҖқxвҖқгҖӮ

зұ»еһӢеҸӮж•°зҡ„ const дҝ®йҘ°з¬Ұ

д»Һ TypeScript 5.0 зүҲжң¬ејҖе§ӢпјҢеҸҜд»Ҙ const еңЁжіӣеһӢзұ»еһӢеҸӮж•°дёҠжҢҮе®ҡеұһжҖ§гҖӮиҝҷеҸҜд»ҘжҺЁж–ӯеҮәжңҖзІҫзЎ®зҡ„зұ»еһӢгҖӮи®©жҲ‘д»¬зңӢдёҖдёӘдёҚдҪҝз”Ё const зҡ„зӨәдҫӢпјҡ

жӯЈеҰӮжӮЁжүҖзңӢеҲ°зҡ„пјҢеұһжҖ§aе’ҢbжҳҜйҖҡиҝҮ зұ»еһӢжҺЁж–ӯеҮәжқҘзҡ„string гҖӮ

зҺ°еңЁпјҢи®©жҲ‘д»¬зңӢзңӢ const зүҲжң¬зҡ„е·®ејӮпјҡ

зҺ°еңЁжҲ‘д»¬еҸҜд»ҘзңӢеҲ°еұһжҖ§ a е’Ң b иў«жҺЁж–ӯдёәconstпјҢеӣ жӯӨ a е’Ң bиў«и§Ҷдёәеӯ—з¬ҰдёІж–Үеӯ—иҖҢдёҚд»…д»…жҳҜ string зұ»еһӢгҖӮ

еёёйҮҸж–ӯиЁҖ

жӯӨеҠҹиғҪе…Ғи®ёжӮЁж №жҚ®еҸҳйҮҸзҡ„еҲқе§ӢеҢ–еҖјеЈ°жҳҺе…·жңүжӣҙзІҫзЎ®зҡ„ж–Үеӯ—зұ»еһӢзҡ„еҸҳйҮҸпјҢиҝҷеҗ‘зј–иҜ‘еҷЁиЎЁжҳҺиҜҘеҖјеә”иў«и§ҶдёәдёҚеҸҜеҸҳж–Үеӯ—гҖӮ иҝҷйҮҢжңүдёҖдәӣдҫӢеӯҗпјҡ

еңЁеҚ•дёӘеұһжҖ§дёҠпјҡ

еңЁж•ҙдёӘеҜ№иұЎдёҠпјҡ

иҝҷеңЁе®ҡд№үе…ғз»„зҡ„зұ»еһӢж—¶зү№еҲ«жңүз”Ёпјҡ

жҳҫејҸзұ»еһӢжіЁйҮҠ

жҲ‘д»¬еҸҜд»Ҙе…·дҪ“ең°дј йҖ’дёҖдёӘзұ»еһӢпјҢеңЁдёӢйқўзҡ„зӨәдҫӢдёӯпјҢеұһжҖ§xзҡ„зұ»еһӢжҳҜnumberпјҡ

жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮдҪҝз”Ёеӯ—йқўйҮҸзұ»еһӢзҡ„иҒ”еҗҲдҪҝзұ»еһӢжіЁйҮҠжӣҙеҠ е…·дҪ“пјҡ

зұ»еһӢзј©е°Ҹ

зұ»еһӢзј©е°ҸжҳҜ TypeScript дёӯзҡ„дёҖдёӘиҝҮзЁӢпјҢе…¶дёӯдёҖиҲ¬зұ»еһӢзј©е°Ҹдёәжӣҙе…·дҪ“зҡ„зұ»еһӢгҖӮеҪ“ TypeScript еҲҶжһҗд»Јз Ғе№¶зЎ®е®ҡжҹҗдәӣжқЎд»¶жҲ–ж“ҚдҪңеҸҜд»Ҙз»ҶеҢ–зұ»еһӢдҝЎжҒҜж—¶пјҢе°ұдјҡеҸ‘з”ҹиҝҷз§Қжғ…еҶөгҖӮ

зј©е°Ҹзұ»еһӢеҸҜд»ҘйҖҡиҝҮдёҚеҗҢзҡ„ж–№ејҸеҸ‘з”ҹпјҢеҢ…жӢ¬пјҡ

жқЎд»¶

йҖҡиҝҮдҪҝз”ЁжқЎд»¶иҜӯеҸҘпјҲжҜ”еҰӮ if жҲ– switchпјүпјҢTypeScript еҸҜд»Ҙж №жҚ®жқЎд»¶зҡ„з»“жһңзј©е°Ҹзұ»еһӢиҢғеӣҙгҖӮдҫӢеҰӮпјҡ

жҠӣй”ҷжҲ–иҖ…иҝ”еӣһ

жҠӣеҮәй”ҷиҜҜжҲ–д»ҺеҲҶж”ҜжҸҗеүҚиҝ”еӣһеҸҜз”ЁдәҺеё®еҠ© TypeScript зј©е°Ҹзұ»еһӢиҢғеӣҙгҖӮдҫӢеҰӮпјҡ

еңЁ TypeScript дёӯзј©е°Ҹзұ»еһӢиҢғеӣҙзҡ„е…¶д»–ж–№жі•еҢ…жӢ¬пјҡ

  • instanceof ж“ҚдҪң: з”ЁдәҺжЈҖжҹҘеҜ№иұЎжҳҜеҗҰжҳҜзү№е®ҡзұ»зҡ„е®һдҫӢгҖӮ
  • in ж“ҚдҪң: з”ЁдәҺжЈҖжҹҘеҜ№иұЎдёӯжҳҜеҗҰеӯҳеңЁеұһжҖ§гҖӮ
  • typeof ж“ҚдҪң: з”ЁдәҺеңЁиҝҗиЎҢж—¶жЈҖжҹҘеҖјзҡ„зұ»еһӢгҖӮ
  • еҶ…йғЁеҮҪж•°пјҢжҜ”еҰӮ: Array.isArray(): з”ЁдәҺжЈҖжҹҘеҖјжҳҜеҗҰдёәж•°з»„гҖӮ

еҸҜеҢәеҲҶиҒ”еҗҲ

дҪҝз”ЁвҖқеҸҜеҢәеҲҶиҒ”еҗҲвҖқжҳҜ TypeScript дёӯзҡ„дёҖз§ҚжЁЎејҸпјҢе…¶дёӯеҗ‘еҜ№иұЎж·»еҠ жҳҫејҸвҖқж ҮзӯҫвҖқд»ҘеҢәеҲҶиҒ”еҗҲеҶ…зҡ„дёҚеҗҢзұ»еһӢгҖӮиҜҘжЁЎејҸд№ҹз§°дёәвҖқж Үи®°иҒ”еҗҲвҖқгҖӮеңЁд»ҘдёӢзӨәдҫӢдёӯпјҢвҖңtagвҖқз”ұеұһжҖ§вҖқtypeвҖқиЎЁзӨәпјҡ

з”ЁжҲ·е®ҡд№үзҡ„зұ»еһӢдҝқжҠӨ

еңЁ TypeScript ж— жі•зЎ®е®ҡзұ»еһӢзҡ„жғ…еҶөдёӢпјҢеҸҜд»Ҙзј–еҶҷдёҖдёӘз§°дёәвҖқз”ЁжҲ·е®ҡд№үзұ»еһӢдҝқжҠӨвҖқзҡ„иҫ…еҠ©еҮҪж•°гҖӮеңЁдёӢйқўзҡ„зӨәдҫӢдёӯпјҢжҲ‘д»¬е°ҶеңЁеә”з”ЁжҹҗдәӣиҝҮж»ӨеҗҺеҲ©з”Ёзұ»еһӢи°“иҜҚжқҘзј©е°Ҹзұ»еһӢиҢғеӣҙпјҡ

switch-true зұ»еһӢзј©еҮҸ

TypeScript 5.3 ж–°еўһдәҶ switch-true зұ»еһӢзј©еҮҸеҠҹиғҪпјҢе…Ғи®ёдҪ дҪҝз”Ёеёғе°”жқЎд»¶пјҢйҖҡиҝҮ switch (true) иҜӯеҸҘжӣҝжҚўз№Ғзҗҗзҡ„ if/else й“ҫгҖӮиҝҷж—ўжҸҗй«ҳдәҶд»Јз Ғзҡ„еҸҜиҜ»жҖ§пјҢеҸҲиғҪе®һзҺ°зұ»еһӢзј©еҮҸгҖӮе®ғзұ»дјјдәҺжЁЎејҸеҢ№й…ҚпјҢдҪҶжӣҙз®ҖжҙҒгҖӮ
